# UI Design / Frontend Skill

Elite frontend UI systems builder. Generates premium-quality SaaS interfaces with perfect visual hierarchy, production-ready code, and accessibility standards.

## When to Use

- Building new UI components or pages
- Designing interfaces from Figma or screenshots
- Creating dashboards, landing pages, auth flows, SaaS UIs
- Improving UI for polish and visual refinement
- Generating pixel-perfect React components

## What It Does

✨ **Copy UIs 1:1** — Generate pixel-perfect React components from designs
🔍 **Self-Validate** — Verify code matches design specifications
🎨 **Premium Quality** — Modern SaaS-level interfaces with excellent UX
⚡ **Production-Ready** — TypeScript + Tailwind CSS + shadcn/ui
♿ **Elite Accessibility** — W3C APG standards, full keyboard support
🎬 **Smooth Interactions** — Motion.dev polish and refinement

## Default Tech Stack

- Next.js / React
- TypeScript
- Tailwind CSS
- shadcn/ui components
- Radix UI (accessibility)
- Motion.dev (animations)

The skill intelligently uses these sources in order:

9. **Tailwind Plus** — Premium quality benchmarks
10. **Vercel Templates** — Real-world SaaS patterns

## Quality Standards

Every generated UI will be:
- ✓ Intentional (purpose-driven decisions)
- ✓ Modern (current patterns, not dated)
- ✓ Crisp (clean, sharp, professional)
- ✓ Well-spaced (breathing room, not cramped)
- ✓ Trustworthy (premium, not generic)
- ✓ Usable (intuitive interactions)
- ✓ Accessible (inclusive, W3C compliant)
- ✓ Cohesive (consistent across pages)

## Never Does

- ✗ Generic AI-looking interfaces
- ✗ Weak spacing or cramped layouts
- ✗ Poor visual hierarchy
- ✗ Inaccessible custom controls
- ✗ Outdated design patterns
- ✗ Gratuitous animation
- ✗ One-off hacks (always reusable)

## Implementation Flow

1. **Determine interface type** (dashboard, landing page, form, etc.)
2. **Gather pattern inspiration** (21st.dev, Tailwind Plus, Vercel)
3. **Build with stack** (shadcn/ui, Tailwind, TypeScript)
4. **Validate correctness** (accessibility, responsiveness, semantics)
5. **Refine for premium feel** (spacing, hierarchy, contrast, polish)
