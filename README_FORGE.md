## What is Forge?

Forge is an autonomous AI agent platform designed for lead generation and business automation. It automates the process of finding, researching, and reaching out to potential clients for service-based businesses.

**Core Features:**
- 🤖 Autonomous AI agents for lead generation
- 📊 Advanced research and data collection
- 💳 Credit-based billing system
- 👥 Team collaboration and management
- 🔑 API access for developers
- 📱 Mobile-responsive design
- 🔔 Real-time notifications
- 📈 Analytics and reporting

## API Endpoints

### Authentication
- `POST /api/auth/signup` - User registration
- `POST /api/auth/login` - User login
- `GET /api/auth/me` - Get current user

### Tasks
- `POST /api/tasks` - Create task
- `GET /api/tasks` - List tasks
- `GET /api/tasks/:id` - Get task details
- `PUT /api/tasks/:id` - Update task

### Credits
- `GET /api/credits/balance` - Get balance
- `GET /api/credits/history` - Get history
- `POST /api/credits/purchase` - Purchase credits

### Teams, Notifications, Settings, API Keys
See FORGE_BACKEND_SPEC.md for complete API documentation.
